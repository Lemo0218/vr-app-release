# VR Basic Calculator

라오어 VR 매매법을 계산하고 사이클별 매수/매도 기준을 확인하는 데스크탑 앱입니다.

## 다운로드

최신 설치 파일은 아래 Release 페이지에서 받을 수 있습니다.

https://github.com/Lemo0218/vr-app-release/releases/latest

## 설치 파일 선택

- Windows: `vr-basic-calculator-0.1.0-windows-x64-setup.exe`
- macOS: `vr-basic-calculator-0.1.0-macos-universal.dmg`
- Linux: `vr-basic-calculator-0.1.0-linux-x64.AppImage`

## 실행 안내

Windows에서 처음 실행할 때 SmartScreen 경고가 뜰 수 있습니다. 이 경우 추가 정보 메뉴에서 실행을 선택하면 됩니다.

macOS에서는 아직 Apple 개발자 인증/공증이 적용되지 않아 확인되지 않은 개발자 경고가 뜰 수 있습니다. 이 경우 파일을 우클릭한 뒤 열기를 선택하거나, 시스템 설정의 보안 항목에서 실행을 허용하면 됩니다.

## 참고

이 저장소는 설치 파일 배포용입니다. 앱 개발 소스코드는 포함하지 않습니다.
